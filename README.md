# MemoApplication_Android
- CRUD 기능을 가진 메모장 앱을 Android와 Flutter에서 똑같은 방식으로 구현합니다.
- 데이터베이스는 Firebase를 사용합니다.
- 완성된 두 앱을 Firebase 성능 모니터링과 Android Studio 프로파일러로 분석합니다.
